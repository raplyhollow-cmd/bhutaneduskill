## Customization Instructions

Each template contains placeholders marked with brackets like `[Date]` or `[Legal Entity Name]`. Before using:

1. Replace all placeholders with actual values
2. Review with Bhutanese legal counsel
3. Customize for your specific business structure
4. Ensure alignment with latest Bhutanese regulations
5. Translate to Dzongkha where required
