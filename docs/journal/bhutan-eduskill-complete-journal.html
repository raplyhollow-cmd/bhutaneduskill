Problems Faced & Solutions

The critical issues encountered and how they were resolved

1

Framer Motion: iterationCount Error

Problem: Using repeat: Infinity without repeatType: "loop" caused "iterationCount must be non-negative" errors.

Solution: Added repeatType: "loop" to all infinite animations. Fixed 6 components.

Files: AssessmentContainer.tsx, QuestionCard.tsx, hero-section.tsx, etc.
2

Database Schema Drift

Problem: 10+ missing tables, 100+ TypeScript errors due to schema-code mismatch. Column name inconsistencies (clerkId vs clerkUserId).

Solution: Created missing tables, standardized column names, fixed all TypeScript errors.

Result: 100+ errors → 0 errors
3

Platform Admin Setup Redirect Loop

Problem: Platform admins were redirected to setup wizard instead of admin dashboard.

Solution: 3-file fix: set-role API returns needsSetup: false for admins, admin layout early-returns, dashboard redirects to /admin.

4

200+ Explicit Any Types

Problem: Widespread use of any type bypassed TypeScript safety.

Solution: Systematic replacement with proper types. Reduced from 241 to ~131 explicit any types.

5

Import Path Inconsistencies

Problem: Mixed use of @/ alias and relative paths (../) causing import resolution errors.

Solution: Standardized all imports to use @/ alias consistently.
