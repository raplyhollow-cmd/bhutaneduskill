# School Management URLs

## Development Server
Running at: http://localhost:3002

## Public Routes
- **Home:** http://localhost:3002/
- **About:** http://localhost:3002/about
- **Contact:** http://localhost:3002/contact
- **FAQ:** http://localhost:3002/faq
- **Sign In:** http://localhost:3002/sign-in
- **Sign Up:** http://localhost:3002/sign-up

## Role-Based Dashboards (require authentication)
- **Student:** http://localhost:3002/student
- **Teacher:** http://localhost:3002/teacher
- **Parent:** http://localhost:3002/parent
- **Counselor:** http://localhost:3002/counselor
- **Admin:** http://localhost:3002/admin
- **School Admin:** http://localhost:3002/school-admin
- **Dashboard:** http://localhost:3002/dashboard

## Note
- School management route is `/school-admin` (not `/school-management`)
- Production URL will be configured on Vercel deployment
